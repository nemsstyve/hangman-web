# hangman & hangman web solutions

## Play hagman-classic
Go into the hangman-classic directory:
```bash
# from root of project
cd hangman-classic
go run main.go ..\words.txt
```

## Play hagman-web
Go into the hangman-web directory:
```bash
# from root of project
cd hangman-web
go run main.go ..\words.txt
```
Then go on http://localhost:8080 to play the game
